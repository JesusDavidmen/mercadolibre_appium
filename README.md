# Automatización Mercado Libre - Appium

Automatiza la búsqueda y filtrado de productos en Mercado Libre usando Appium y Ruby.

## Cómo ejecutar

1. Clonar el repositorio:
git clone https://github.com/JesusDavidmen/mercadolibre_appium.git
cd mercadolibre_appium

2. Instalar dependencias:
bundle install

3. Ejecutar el script:
bundle exec ruby mercadolibre_test.rb
